Shiny advantages and cons

Advantages

  • Shiny is one of the easiest ways to develop useful and data based applications without web development knowledge.
  • Support online demos
  • Empowered by Javascript and CSS!

Cons

  • Shiny server have tricky details in the installation and configuration process.
  • R in the server package installation have some permission issues if you do not have root access.
  • Debugging issues at the server side.

Server code

require(shiny)

## Loading required package: shiny

require(rCharts)

## Loading required package: rCharts

shinyServer(function(input, output, session){

    observeEvent(input$SubmitButton, {

         output$minibuffer <- renderText("*** Loading Twitter data, please wait...:")

         output$map_container <- renderMap({
             output$minibuffer <- renderText("*** Network map for:")
             output$minibufferUser <- renderText(input$userName)
             plotMap(input$userName)

     })
 })
})
